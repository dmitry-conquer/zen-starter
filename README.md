## 🚀 Getting Started

### 1. Create Project
```bash
npm create zen@latest
cd my-project
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Start Development
```bash
npm run dev
```

### 4. Open Browser
Navigate to `http://localhost:5173`
