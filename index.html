ZEN Starter
A premium Vite starter kit for rapid web development
✨Features
  • 🌀 Full Lenis Smooth Scroll support
  • 🗂️ Ready-to-use Tabs, Modals, Accordions, Theme Switcher
  • 📦 Clear and intuitive project structure
  • 🔄 Continuous dependency updates & support
  • 🎨 Prettier, ESLint, Autoprefixer
  • 💎 SCSS/Sass with variables, mixins, helpers
  • 🛡️ TypeScript for scalable code
  • 🧩 Handlebars templating
  • ⚡ Modern build with Vite
  • 🔌 PostCSS & plugin ecosystem
  • 🚀 Ready for production and rapid prototyping
⚡Quick Start Commands
  • npm install – Install dependencies
  • npm run dev – Start development server
  • npm run build – Build for production
  • npm run preview – Preview production build
  • npm run lint – Lint TypeScript files
  • npm run lint:fix – Lint & auto-fix TypeScript files
