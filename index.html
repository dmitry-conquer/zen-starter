üíéSCSS Documentation

Comprehensive SCSS architecture with utilities, mixins, and functions for scalable styling:

Available Mixins

  • @mixin font-face($font-family, $file-name, $font-weight, $font-style) - Define custom font faces with WOFF2 support
  • @mixin hide - Hide element with opacity and visibility
  • @mixin show - Show element with opacity and visibility
  • @mixin abs-center - Center element absolutely using transform
  • @mixin flex-center($isInline) - Center content with flexbox (inline or block)

SCSS Functions

  • rem($pixel) - Convert pixels to rem units (16px base)
  • round-to($number, $decimals) - Round number to specified decimal places
  • fluid($min, $max) - Create fluid typography with clamp()

Usage Examples

// Using mixins
.button {
  @include flex-center;
  @include abs-center;
  
  &.hidden {
    @include hide;
  }
  
  &.visible {
    @include show;
  }
}

// Using functions
.container {
  padding: rem(20);
  font-size: fluid(16, 24);
  border-radius: round-to(3.14159, 2);
}

// Font face definition
@include font-face("CustomFont", "custom-font", 400, normal);
